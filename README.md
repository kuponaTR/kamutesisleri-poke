# Poke Integration for kamutesisleri.com

Bu proje `kamutesisleri.com` sitesini tarayıp Notion veritabanına kayıt ekleyen basit bir örnek entegrasyondur. Poke veya benzeri bir araç GitHub üzerinden çekip çalıştırabilir.

Kurulum

1. Node 18+ yüklü olduğundan emin olun.
2. `.env` dosyasını `.env.example`'den kopyalayıp doldurun.
3. `npm install` ardından `npm run scan` çalıştırın.

NOT: Gerçek kullanım için OAuth kimlik bilgilerini ve tokenları güvenli şekilde saklayın. Bu repo örnek amaçlıdır.
